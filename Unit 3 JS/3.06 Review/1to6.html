  2. What is the difference between a variable and a constant? Give an example of how you would define one of each in JS.
    • A variable is a container that can vary in what it holds over the course of the program, whereas a constant stays the same and can't be changed.
    • var x = 7; // Variables can be defined with keyword 'var'
let y = -3; // Variables can also be defined with 'let'
const PI = 3.14159265; // Constants are defined with 'const' and their names should be capitalized
  3. What is the difference between the data type Number and String? Give an example of each.
    • Numbers are like numbers in math, in that they can be integers, negative, decimal (called a floating point number) or they could be 0. You can perform arithemtic operations on Numbers like you can in math.
      • const TEMP = -7.3;
const FEELS_LIKE = TEMP*APPARENT_TEMP_COEFF;
    • Strings are an array of characters that can form words, sentences, passwords, etc. Strings can also be added together, but when we do this they're literally just put together which is called concatenation. There are also other operations for Strings including converting the case to all caps or all lowercase. The following is a definition of a string literal:
      • const USER = "dingus123";
